# Terminal Messenger (Java)

**Terminal Messenger** — это терминальный чат на Java для общения в сети.  
---

## Функционал

- Обмен текстовыми сообщениями между двумя ПК в одной сети
- Отдельный поток для получения сообщений
- Конфигурация IP и порта через `config.properties`

---
